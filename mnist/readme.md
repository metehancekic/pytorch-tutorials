## 4 Layer CNN
A simple tutorial of MNIST and Fashion MNIST classification with 4 layer convolutional neural network (CNN). Neural network is implemented with torch modules.

**Libraries Needed**

- Python			3.6
- Torch 			1.10.0
- Torchvision		0.2.2
- Matplotlib 		3.0.3
- Numpy			1.16.2
- Tqdm (Optional) 	4.31.1
- Argparse			1.1
